# dom操作
> 使用方式是导入`domUtils`模块,主要是包括一些简单的dom操作方法,更复杂的操作可以使用`@vueuse/core`;
## methods
- `{Fn} getBoundingClientRect`：获取边距距离
- `{Fn} hasClass`：判断包含类
- `{Fn} addClass`：添加类
- `{Fn} removeClass`：删除类
- `{Fn} getViewportOffset`：获取Offset
- `{Fn} on`：事件
- `{Fn} off`：事件
- `{Fn} once`：事件
- `{Fn} setCssVar`：设置css var
- `{Fn} getCssVar`：获取css var
