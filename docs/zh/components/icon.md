##  属性 


| 属性    | 说明   | 类型       | 可选值 | 默认值 |
|-------|------|----------|-----|---|
| icon   | 名称   | `string` | -   |  |
| color  | 颜色   | `string` | -   |   | 
| size | 尺寸   | `number` | -   | `16` | 
| spin | 加载效果 | `boolean` | -   | `false` | 
| prefix | 前缀   | `string` | -   | `''` |
